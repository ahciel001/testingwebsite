General Item Rule Changes
  • Tier 2 items are now restricted to Tier 2 (level 5 and above) characters. If you currently have an item that is beyond your level because of this rule, you keep it but cannot use it until you reach Tier 2.
  • The Potion of Invulnerability tier and price has been changed.
  • Any item that has a recharge time longer than 10 Days is no longer rechargeable and is now considered a consumable.
